# Wave Music

Wave Music é um player local para Android feito com Kotlin, Jetpack Compose e Material 3.

## Funcionalidades

- Lê músicas baixadas no dispositivo usando MediaStore.
- Toca áudio real com MediaPlayer.
- Tela inicial com lista de músicas.
- Busca por música, artista ou álbum.
- Mini player fixo e tela Tocando agora.
- Músicas curtidas persistidas no aparelho.
- Playlists persistidas no aparelho.
- Biblioteca com Curtidas, Playlists, Álbuns e Artistas.
- Tema escuro com visual neon em roxo, azul e rosa.

## Como rodar

1. Abra o projeto no Android Studio.
2. Aguarde o Gradle Sync.
3. Selecione um celular ou emulador.
4. Clique em Run.
5. Aceite a permissão de áudio no app.

O APK debug também está em `release/WaveMusic-debug.apk`.
